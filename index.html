Feed Manipulation Algorithm

This website serves to demonstrate simplified social media algorithm processes. This website displays the feed-manipulation seen on social media, through algorithmic processes determining the user's prefference for cats or dogs. When the user engages with the website, the digital profile of the user, will adapt to display the users prefference for cats or dogs. As the profile begins determining the user's prefference for cats or dogs, it will begin to lessen the likelyhood of displaying the least preffered animal.
